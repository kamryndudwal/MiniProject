Using SVG Edit

This demo will show you how to use SVG Edit.

  • SVG edit can be used to create new graphics, and edit/enhance a photo or image.
  • You will create an image like the one below and then test to ensure both images are the same.
  • First click on the rectangle on the left sidebar and draw a rectangle on the canvas.
  • Now ensure the x and y, located on the top toolbar when the rectangle is clicked, has the values of x=80 and y=100 and w=300 and h=150
  • Now click on the circle icon in the left sidebar and draw a circle/ellipse on the canvas.
  • Now ensure the cx and cy, located on the top toolbar when the rectangle is clicked, has the values of cx=500 and cy=350 and rx=90 and ry=90
  • Changing the color, click on the shape then click the bucket in the lower left corner
  • change the color of the rectangle to a light blue and the circle to a pale yellow
  • Save your image by clicking svg-edit in the top left corner and then save, then add it to the folder containing this code.
  • Upload your image into the selected place in the html code
  • Now use the slider to ensure your image is the same as the image that already appears.
  • There is an example image with red shapes that shows how it works and how to recognize the similarities.
  • To help me create the slider I used information from this website.

Using HTML5 Canvas

Click the draw button to see a face being made.

Then click the button for each of the different facial features to choose what color each should be.

Remember to press DRAW before changing the color of the eyes or nose.

When a button is pressed, this sends the HTML to the JS and computes the JS, in this case the JS is using the canvas that was made in HTML todraw the head, using .arc, then the eyes and nose also using .arc.

The mouth uses .quadraticCurveTo in order to create a smiling effect. The colors follow the traditional rgb format.

Your browser does not support the HTML canvas tag.

Animation in HTML5 Canvas

This is a very simple animation of a ball going across the screen however it is very important to understand what is going on because this type of code and animations are used in many different places like games and website development.

In this code the balls are drawn the same way the face above was. To get the ball to move the x position value is being incremented by a certain value until the ball reaches the end of the page.

We know it isn't on the page by using a simple if statement to check the position of the balls.
